## Setup
create app
```
  create-react-app .
```
Install redux libraries
```
  npm i -S redux react-redux
```
Install UT libraries
```
  npm i -D enzyme enzyme-adapter-react-16
```
Create src/setupTests.js to config enzyme. Jest will find and use it.
